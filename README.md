# vue3-vite-composition-crud

Aplicacion web de inventario de libros creada con Vue 3 y Vite. Esta primera etapa contiene solamente la configuracion inicial del proyecto.

## Tecnologias iniciales

- Vue 3
- Vite
- JavaScript
- Composition API

## Instalacion

```bash
npm install
```

## Ejecucion local

```bash
npm run dev
```

## Build de produccion

```bash
npm run build
```
