# vue3-vite-composition-crud

Aplicacion web de inventario de libros creada con Vue 3 y Vite. En esta etapa se configura Vue Router, JSON Server y el primer listado de libros usando Axios.

## Tecnologias iniciales

- Vue 3
- Vite
- JavaScript
- Composition API
- Vue Router
- Axios
- JSON Server

## Instalacion

```bash
npm install
```

## Ejecucion local

```bash
npm run dev
```

## API local con JSON Server

Levantar la API:

```bash
npm run server
```

URL de libros:

```text
http://localhost:3000/libros
```

## Build de produccion

```bash
npm run build
```
